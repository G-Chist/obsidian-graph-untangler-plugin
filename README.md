# obsidian-graph-untangler-plugin

This plugin for [Obsidian](https://obsidian.md/) uses Forces to automatically untangle your graph for you.

![Untangling animation](images/the_untangling.gif)

## Local Installation

To install this plugin locally, copy the files from this repository into YOUR_VAULT_NAME/.obsidian/plugins/graph-untangler/

In OBsidian, go to Settings -> Community Plugins -> Installed Plugins, then click Reload plugins and enable Graph Untangler.

After enabling the plugin, you may set hotkeys for untangling and stopping untangling.
